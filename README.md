
# Royal Rangers Outpost 5

  

## Installation Instructions

1. Clone Repo
2. Run ``npm install``
3. Split Terminal or Open a second terminal tab
4. Run ``npm run dev-server`` in one tab **(Must be run first)**
5. Run ``npm start`` in the other
